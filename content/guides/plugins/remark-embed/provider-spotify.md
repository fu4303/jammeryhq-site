## Configuration

| Name   | Type     | Description                            | Allowed Values  | Default |
| ------ | -------- | -------------------------------------- | --------------- | ------- |
| width  | `string` | Sets the maximum width for the iframe  | `400px`, `100%` | `300px` |
| height | `string` | Sets the maximum height for the iframe | `400px`, `100%` | `380px` |
